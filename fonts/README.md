```html
<link href="https://fonts.googleapis.com/css?family=Irish+Grover|Joti+One|Kirang+Haerang|Lakki+Reddy|Ravi+Prakash|Risque&display=swap&subset=korean,latin-ext,telugu" rel="stylesheet">
```

```css
font-family: 'Joti One', cursive;
font-family: 'Irish Grover', cursive;
font-family: 'Ravi Prakash', cursive;
font-family: 'Lakki Reddy', cursive;
font-family: 'Risque', cursive;
font-family: 'Kirang Haerang', cursive;
```

```css
@import url('https://fonts.googleapis.com/css?family=Irish+Grover|Joti+One|Kirang+Haerang|Lakki+Reddy|Ravi+Prakash|Risque&display=swap&subset=korean,latin-ext,telugu');
```

```html
<link href="https://fonts.googleapis.com/css?family=Lobster|Open+Sans:400,400i,700,700i&display=swap&subset=cyrillic,cyrillic-ext,latin-ext,vietnamese" rel="stylesheet">
```

```css
@import url('https://fonts.googleapis.com/css?family=Lobster|Open+Sans:400,400i,700,700i&display=swap&subset=cyrillic,cyrillic-ext,latin-ext,vietnamese');
```

```css
--font-body: "Open Sans", sans-serif;
```
